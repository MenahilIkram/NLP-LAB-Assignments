# Mental Health and Psychological Disorders Dataset

##  Project Overview

This project focuses on building a custom NLP dataset using web scraping techniques from Wikipedia.

##  Objective

To collect, process, and organize textual data related to mental health conditions and psychological disorders.

##  Steps Performed

### Step 1: Data Extraction

* Collected data from Wikipedia using Python
* Extracted content for multiple mental health topics
* Saved each topic as an individual text file

### Step 2: Data Merging

* Combined all individual files into a master dataset
* Created both TXT and CSV formats

### Step 3: GitHub Upload

* Uploaded all code and dataset files to this repository

### Step 4: Kaggle Upload

* Published dataset on Kaggle for public use
* You can access the dataset here:
* https://www.kaggle.com/datasets/menahilikram/mental-health-and-psychological-disorders-dataset
